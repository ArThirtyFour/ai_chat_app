# AI Chat App

Описание :
__Приложение , которое позволяет через приложение доставать информацию через нейросеть.__

Модули:
> _Python 3.11_
> 
> _Tkinter_
> 
> _OpenAI_

Установка:
```
git clone https://github.com/ArThirtyFour/ai_chat_app
cd ai_chat_app
pip install tk openai
python file.py
```
